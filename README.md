This is a Hotel Management System built using PHP, MySQL, HTML, CSS, and JavaScript. The system helps hotels manage bookings, rooms, users, and payments, while providing customers with an easy booking experience.

Features

User Side:

User registration and login

Browse hotels and room types

Check room availability

Book rooms with check-in/check-out dates

Online payment options

Manage profile and booking history

Admin Side:

Dashboard with booking overview

Manage hotels, rooms, and pricing

View and manage bookings

Manage users and guest details

Generate invoices and PDF reports

Session management with timeout

Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: PHP (Core PHP, PDO)

Database: MySQL

Other Tools: PHPMailer, FPDF (optional)
